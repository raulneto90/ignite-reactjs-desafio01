## :information_source: Informações do projeto

Sistema de gerenciamento de tarefas (Todo's em inglês) onde o usuário pode:

- [X] Cadastrar uma nova tarefa
- [X] Remover uma tarefa cadastrada
- [X] Marcar a tarefa como concluída

Este sistema foi desenvolvido a partir do desafio 01 da Trilha ReactJS do Ignite, da Rocketseat.

## :floppy_disk: Como utilizar o projeto

Para baixar e utilizar o projeto:

```bash
$ git clone https://github.com/raulneto90/ignite-reactjs-desafio01
$ cd ignite-reactjs-desafio01

# Instalar as dependências do projeto
$ yarn

# Rodar o projeto em modo desenvolvimento
$ yarn dev
```
